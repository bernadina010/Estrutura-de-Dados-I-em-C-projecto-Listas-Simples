1º Trabalho prático de Estrutura de Dados I
Objectivo: Prática de Tipos Abstractos de Dados e Lista Encadeadas Simples (8Valores)

1. Um cliente pode ser definido por: número, nome, telefone e data de nascimento. Usando listas encadeadas simples, implemente o TAD lista de clientes com pelo menos 5 operações básicas à escolha do grupo,
sabendo que as operações escolhidas devem permitir a manipulação da lista. Para além das operações básicas devem também criar as seguintes:
  
  a. Faça uma função que remove repetições da lista, ou seja, clientes com o mesmo número. 
  
  b. Cria uma função imprima os dados dos clientes por ordem da data de nascimento. 

Deve ser criado um programa com um menu onde aparecem todas as funções implementadas, para teste.
